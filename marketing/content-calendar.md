# KithNode Content Calendar + Weekly System (Phase 0)

*The operating engine. ~5 hr/week, two channels, everything to the waitlist.*

## The repurposing loop (do this once per week)
Pick ONE core idea from the want-vs-hate research (e.g. "warm intros reply 5x more than cold"). Turn that single idea into:
- 1 Reddit value comment or post (no product)
- 1-2 faceless TikTok scripts (screen-record or text-story)
- (Phase 1) 1 LinkedIn post

One idea, many formats. Never start a post from a blank page.

## Weekly schedule (~5 hr total)

| Day | Action | Time |
|-----|--------|------|
| Mon | Batch block: pick the week's idea, draft 3-4 TikToks + 1 Reddit value post + 1 build-in-public post (use marketing skills) | ~90 min |
| Tue | Post TikTok #1; 2-3 Reddit value comments; post the value post | ~30 min |
| Wed | Post TikTok #2; 2-3 Reddit value comments | ~25 min |
| Thu | Post TikTok #3; build-in-public post (r/EntrepreneurRideAlong); reply to comments | ~30 min |
| Fri | Post TikTok #4; 2-3 Reddit value comments | ~25 min |
| Sat | Light: reply to any comments/DMs, save anyone who asks to the waitlist | ~15 min |
| Sun | Fill `tracking-log.md`: what shipped, reach, waitlist adds, pick next week's idea | ~20 min |

## Rules
- Post over perfect. A B-plus post shipped beats an A-plus post that never goes out.
- Account warming first on Reddit (see reddit-playbook.md) before any promo-adjacent post.
- Every post has a path to the waitlist (bio link, pinned comment, or soft CTA).
- Batch production, real-time engagement. Draft Monday, engage live all week.

## Idea bank (seed from the research; add as you go)
- Warm intros reply 5x more than cold.
- Follow-ups generate ~42% of replies; day 5-7 is the window.
- Shared signal in the subject line (school/hometown/club) flips the open rate.
- Cap touches at 2-3; persistence past that reads as desperation.
- The spreadsheet's only failure mode: it never tells you who's overdue.
- AI that auto-sends is slop; AI that researches + drafts is leverage.
