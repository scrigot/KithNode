# marketing/queue/

Build-in-public content drafts produced by `/kithnode-content-engine`. One file per run: `<YYYY-MM-DD>-<slug>.md`, holding one mined idea repurposed into a Reddit post + 1-2 faceless TikTok scripts (+ LinkedIn in Phase 1).

Rules:
- Drafts ONLY. Sam approves and posts by hand. Nothing here is auto-published.
- Honesty-clamped: every number traces to a real `ops/build-log.md` / `metrics.ts` value, or it is not there. No em dashes, no emojis (see `ops/ETHOS.md`).
- Each draft carries a `review: PASS` line from the content-engine's separate review pass before it is considered ready.
